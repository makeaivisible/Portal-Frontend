# Security Policy

Make AI Visible handles sensitive research workflows involving youth AI conversations. Please report security or privacy concerns privately before public disclosure.

## Report a Vulnerability

Open a private security advisory when available, or contact the project maintainers through the official Make AI Visible contact channel.

## Sensitive Data Rule

Do not include raw conversations, credentials, personal identifiers, school names, addresses, usernames, or private contact details in issues, pull requests, fixtures, screenshots, or logs.

## Scope

Security reports may include vulnerabilities in upload flows, anonymization, reviewer access, scoring data handling, dataset publication, or dashboard privacy thresholds.
